# Contributing

Thank you for your interest in NVBar!

**This is a personal project and I am not accepting contributions at this time.**

The repository is public for transparency and educational purposes. You're welcome to:
- Fork the project for your own use
- Report bugs via issues
- Use the code as reference

However, please note that **pull requests will not be reviewed or merged**.

If you'd like to suggest a feature or report a bug, feel free to open an issue, but understand that I may not be able to address it.
